# Consulting App

## Overview

Consulting App is a full-stack platform that connects clients with consultants.

The platform allows users to:

- Register and authenticate
- Schedule consultations
- Manage payments
- Submit reviews
- View analytics
- Interact with an AI-powered chatbot

---

# Project Structure

/frontend
Backend-independent user interface built with Next.js.

/backend
REST API built with FastAPI.
